Classes

  • main__container: must be called with col-xx row-xx for sizing.
  • main__container-head: container header with background color.
  • main__container-subhead: subheading with background, washed out text color.
  • main__container-text: basic text box with padding, colorization.
  • main__container-table-h: table formatting for horizontal tables.
  • main__container-table-v: table formatting for vertical tables.
  • main__container-image: center formatted image.
  • main__container-image-footnote: italicized, muted text for below image.

Positioning

  • col-xx: grid column span number - each column 50px, 30 columns in total.
  • row-xx: grid row span number - each row 50px, 30 rows in total.
  • pX-X: padding - px, py, pt, pb, 1-5. ex. pt-4
